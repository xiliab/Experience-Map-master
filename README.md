# Experience Map | 体验地图

![](assets/screenshot.jpg)

这是 **LLXiao** 的个人体验地图，一个基于 Web 的可交互作品集展示项目。它以地图的形式展示了设计师的工作背景、项目经历和生活点滴。

## 🌟 核心功能

- **交互式地图导航**：通过滚动页面，在类似“赛车跑道”的路径上浏览设计师的职业生涯。
- **动态进度展示**：实时显示当前年份进度和对应的地理位置（广州等）。
- **多媒体弹窗预览**：支持项目图片、GIF 动画和视频作品的无缝预览，内置圆点指示器，支持翻页切换。
- **深色模式支持**：适配系统主题，支持手动切换深色/浅色模式。
- **响应式设计**：完美适配移动端和桌面端浏览体验。
- **备案信息集成**：集成工信部及公安备案信息展示。

## 🛠️ 技术栈

- **前端**：Vue.js (2.6.11), Vanilla CSS, SVG 路径动画。
- **后端**：PHP (用于环境配置和页面渲染)。
- **字体**：得意黑 (Smiley Sans)。

## 🚀 本地运行

1. 确保本地安装了 PHP 环境。
2. 将项目克隆或解压到服务器根目录。
3. 访问 `index.php` 即可启动。

```bash
# 如果使用 PHP 内置服务器
php -S localhost:8000
```
